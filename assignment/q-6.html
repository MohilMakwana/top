Quest6. What do you understand by the universal sector explain with examples?

The Universal selector can be denoted by an asterisk (*), which is a special CSS selector that targets all elements on a web page. 

        Eg.
        /* Applying styles to all elements using the universal selector */
        * {
          margin: 0;
          padding: 0;
          box-sizing: border-box;
        }
        
        /* Applying a border to all elements for illustration purposes */
        * {
          border: 1px solid #ccc;
        }
